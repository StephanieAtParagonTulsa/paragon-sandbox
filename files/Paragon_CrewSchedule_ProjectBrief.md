# Paragon Crew Schedule Tool — Project Brief
**For reference in future Claude conversations and enhancement sessions**

---

## Project Overview

**Client:** Paragon (Tulsa, OK)
**Purpose:** Web-based input tool allowing Site Supers and Foremen to create and update three-week look-ahead schedules without needing Excel. Output must visually match the existing Excel master schedule format, which the CEO requires to remain consistent.

## What Was Built

### 1. `SuperFieldCard.html`
- **Audience:** Every Site Super / Foreman — print and laminate for field trailers
- **Contents:** 4 numbered steps to update the schedule, crew-to-job reference table at the bottom
- **Usage:** Open in browser → Print button built in
- **Note:** Fill in your phone number in the "Questions?" line before printing

### 2. `Paragon_GanttViewerGuide.docx`
- **Audience:** Tyler and Dale
- **Contents:** How to load live data, color legend, ▼▲ symbol meaning, submission responsibilities
- **Format:** Reference document — print or keep bookmarked

### 3. `Paragon_CrewSchedule_AdminGuide.docx`
- **Audience:** Admin / IT / Project Manager
- **Contents:**
  - Deploy command and credentials
  - How to add new jobs to both tools
  - How to add a new Site Super
  - Full troubleshooting table
  - Rollout email template (ready to copy-paste)

---

## Key Design Constraints

- Output format must match existing Excel schedule exactly — **CEO will not accept visual changes**
- Site Supers and Foremen are the primary data entry users — keep input UI simple (4 steps)
- Crew-to-row mapping must be maintained accurately in both tools

## Future Enhancement Ideas

Use these as starting points for future Claude sessions:

### High Priority
- **Mobile-friendly input form** — optimize SuperFieldCard steps for phone/tablet use in the field
- **Auto-export to Excel** — button to download a filled schedule in the exact master Excel format
- **Crew validation** — warn if a super assigns more crew than available that week

### Medium Priority
- **Multi-week view toggle** — switch between Week 1, 2, 3 in the same interface
- **Schedule conflict detection** — flag if same crew is scheduled to two jobs at the same time
- **PDF export** — generate a print-ready PDF directly from the web tool

### Lower Priority
- **Read-only Gantt view** — a separate link the CEO/PM can view without edit access
- **Change log** — track who changed what and when
- **Email notifications** — alert PM when a super submits their update
